## 📋 Overview

This repository contains my solution for the **Ecomexperts Shopify Developer Hiring Test**.

The task was to build a **pixel-perfect Shopify page** from a Figma design using **two completely custom Shopify sections**, without using any of Dawn's built-in sections or components.

Everything was developed from scratch using:

- Shopify Liquid
- HTML5
- CSS3
- Vanilla JavaScript (No jQuery)

---

## ✅ Requirements Completed

### Banner Section

- Built completely from scratch
- Editable content through Shopify Theme Customizer
- Responsive layout
- Custom animated buttons
- Separate banner image and content section

### Product Grid Section

- Built from scratch
- Displays six selectable products
- Products configurable through Theme Customizer
- Responsive desktop and mobile layouts

### Quick View Popup

- Opens when clicking the product hotspot/button
- Dynamically loads:

  - Product image
  - Product title
  - Price
  - Description
  - Variant options

- Variant selection updates dynamically
- Functional **Add to Cart** button using Shopify AJAX Cart API

### Cart Logic

Implemented the required custom functionality:

- If the selected variant is:

  - **Color:** Black
  - **Size:** Medium

- The product **Soft Winter Jacket** is automatically added to the cart.
